# Fura Handbók

Innri handbók fyrir vélar og tæki hjá Fura Málmendurvinnslu — svo nýtt starfsfólk geti lært vinnuna sína vel.

## Bygging
- **Staðir / vélar** á forsíðu (t.d. Tætarinn, Furuflís, Skemman).
- Hver vél getur innihaldið aðrar vélar/hluti — eins djúpt og þarf.
- Hver síða: myndir frá öllum hliðum með skýringum, bilanaleit, varahlutir, athugasemdir.
- **Leit** (🔍) til að finna vél eða hlut eftir nafni.

## Aðgangur
Tveir 4-stafa kóðar:
- **Lestrarkóði** — les handbókina.
- **Breytingakóði** — má líka bæta við og laga (að hámarki 3 manns).

## Staða
Prufuútgáfa (frumgerð). Gögn vistast á tækinu (localStorage).
Næsta skref: tengja við Supabase fyrir sameiginlega handbók á netinu, alvöru myndahleðslu og uppsetningu í símann (PWA).

## Keyra
Opnaðu `index.html` (eða `Opna handbók.command` á Mac). Plain HTML/JS, engin uppsetning.
